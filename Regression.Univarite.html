Module Regression.Univarite

module Univarite: sig .. end
Simple one dimensional regress.
include Regression.LINEAR_MODEL
val alpha : t -> float
alpha t a shorthand for the constant parameter used in the regression. Equivalent to (coefficients t).(0)
val beta : t -> float
beta t a shorthand for the linear parameter used in the regression. Equivalent to (coefficients t).(1)
val confidence_interval : t -> alpha:float -> input -> float * float
confidence_interval linear_model alpha x Use the linear_model to construct confidence intervals at x at an alpha-level of significance.
val prediction_interval : t -> alpha:float -> input -> float * float
prediction_interval linear_model alpha x Use the linear_model to construct prediction intervals at x at an alpha-level of significance.
