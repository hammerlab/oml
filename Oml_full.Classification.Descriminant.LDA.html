module LDA: 
functor (D : Oml.Classification.Input_interfaces.Continuous_encoded_data) -> sig .. end
Train a Multiclass LDA model by estimating a mean vector for the features (per class) and a common covariance matrix (common to all classes), which are then used to model a Multivariate normal distribution. These, per class, distributions are used in Bayes's rule for classification.
