Module Statistics.Sampling.Poly

module Poly: sig .. end
Provides polymorphic versions that sample over arrays of any type.
val uniform : ?seed:int array -> 'a array -> 'a Statistics.Sampling.generator
uniform ?seed elems creates a generator that will sample from the elems array using the uniformly random distribution.
Raises Invalid_argument if the given element array is empty.
val multinomial : ?seed:int array ->
       'a array -> float array -> 'a Statistics.Sampling.generator
multinomial ?seed elems weights creates a generator will sample from the elems array using Multinomial distribution given by a weights vector which sums to 1.
Raises Invalid_argument if weights do not sum to 1.0 or the length of the elems and weights arrays are not equal.
val softmax : ?seed:int array ->
       ?temperature:float ->
       'a array -> float array -> 'a Statistics.Sampling.generator
softmax ?seed ?temperature elems weights creates a generator that will sample from the elems array using the softmax distribution given by a weights vector and temperatureerature parameter which defaults to 1.0.
Raises Invalid_argument if weights is empty or the length of the elems and weights arrays are not equal.
