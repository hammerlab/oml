Module type Classification.Intf.Generative

module type Generative = sig .. end
A generative classifier builds models of the form P(class, feature).

For current purposes these classifiers can return individual probabilities of the form P(feature | class).

include Classification.Intf.Classifier
val class_probabilities : t -> clas -> float * (feature -> float array)
class_probabilities t class returns the prior and per feature likelihood probabilities learned by t for class.
Raises Not_found if t never trained on class.
