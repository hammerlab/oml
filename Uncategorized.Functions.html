module Functions: sig .. end

Implementations of special functions such as those needed to compute probability distributions.

Implementations of special functions such as those needed to compute probability distributions.

At the present moment many functions are wrappers to the Cephes library (http://www.netlib.org/cephes/), via a Ctypes interface implemented in the Ocephes library.

Standard error functions.


val erf : float -> float
erf x computes the (Gauss) error function, (2/sqrt(pi) * \int_0^x e^-t^2 dt)
val erfc : float -> float
erfc x computes 1.0 - erfc x.

Gamma functions.


val gamma : float -> float
gamma x computes the gamma function of x. For positive integers gamma x approximates (x - 1)!.
val ln_gamma : float -> float
ln_gamma x compute the natural logarithm of the gamma function of x.

It is usually more accurate to use ln_gamma instead of gamma and afterwards compute the exponent.

val regularized_lower_gamma : a:float -> float -> float
regularized_lower_gamma a x computes regularized (normalized by gamma a) incomplete lower (integral from 0 to x) function.
val regularized_upper_gamma : a:float -> float -> float
regularized_upper_gamma a x computes regularized (normalized by gamma a) incomplete upper (integral from x to infinity) function.

Beta functions.


val beta : float -> float -> float
beta x y computes the beta function of x and y, this function is also known as the Euler integral of the first kind and is useful in defining various distributions.
val ln_beta : float -> float -> float
ln_beta x y computes log (beta x y), for more accuracy.
val regularized_beta : alpha:float ->
       beta:float -> ?epsilon:float -> ?max_iterations:int -> float -> float
regularized_beta ?epsilon ?max_iterations ~alpha ~beta x computes the regularized (divided by beta alpha beta) incomplete beta function, which is the partial (0 to x) integral of the beta function paramterized by alpha and beta.
val multivariate_beta : float array -> float
multivariate_beta arr computes the multivariate_beta function arr, this function is also known as the Euler integral of the first kind and is useful in defining various distributions.
val ln_multivariate_beta : float array -> float
ln_multivariate_beta x y computes log (multivariate_beta x y), for better accuracy.
val softmax : ?temperature:float -> float array -> float array
softmax ?temperature weights transforms weights into softmax weights dependent on temperature.
Raises Invalid_argument if weights is empty or temperature = 0.

Distribution CDF's


val chi_square_less : int -> float -> float
chi_square_less k x computes the probability of seeing a value less than x in a Chi-square distribution with k degrees of freedom.
val chi_square_greater : int -> float -> float
chi_square_greater k x computes the probability of seeing a value greather than x in a Chi-square distribution with k degrees of freedom.
val normal_cdf_inv : float -> float
normal_cdf_inv x returns the value y such that the integral of the normal cdf is x.
val student_cdf_inv : degrees_of_freedom:int -> float -> float
student_cdf_inv k x returns the value y such that the integral of the Students T distribution with k degrees of freedom is x.
val f_less : d1:float -> d2:float -> float -> float
f_less d1 d2 x computes the probability of seeing a value less than x in an F-distribution parameterized by d1 and d2.
