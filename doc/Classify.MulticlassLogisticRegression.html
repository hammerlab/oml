Functor Classify.MulticlassLogisticRegression

module MulticlassLogisticRegression: 
functor (D : Continuous_encoded_data_intf) -> sig .. end
Use Multiple Class Logistic Regression to estimate log-odds for each of the quantitative features in the encoded data, per class.

A constant 1 is added to all encoded features by estimate, there is no need to add one with encoding.

Parameters:
D : Continuous_encoded_data_intf 
include Classify.Classifier_intf
val coefficients : t -> float array array
coefficients t the weights assigned to the log-odds of the features.

The first coefficient is for a constant term and therefore there will always be 1 more than the size of features.

val class_order : t -> clas list
class_order t specifies the order in which coefficients are returned.
