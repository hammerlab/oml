Module type Classify.Category_encoded_data_intf

module type Category_encoded_data_intf = sig .. end
Data where features are encoded using a set of Category variables.

Each feature is described using an array of category variables constrained by encoding_sizes.

include Classify.Data_intf
val encoding : feature -> int array
encoding f returns an array where each element e at position i describes the category value e of the ith category variables.
Raises Invalid_argument if the returned array is not of the same length as encoding_sizes or if the ith element takes a greater than or equal to the ith sizes in encoding_sizes.
val encoding_sizes : int array
encoding_sizes the number of categories for each variable.
