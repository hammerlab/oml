type mean_update = size:int ->
       n_sum:float -> n_sum_sq:float -> n_size:float -> t -> float -> float
Defines a mean update function. Given an existing running statistic t, number of samples size, the already computed updated statistics n_sum, n_sum_sq, n_size, and the newly observed value, return a new estimate of the mean.
type var_update = n_mean:float ->
       n_sum:float -> n_sum_sq:float -> n_size:float -> t -> float
Defines a variance update function. Given an existing running statistic t, the already computed updated statistics n_mean n_sum, n_sum_sq, and n_size, return a new estimate of the variance.
val empty : t
empty an empty t, useful for initializing the fold.
val init : ?size:int -> float -> t
init ?size x initializes a t with x and initial size which defaults to 1.
val update : ?size:int ->
       ?mean_update:mean_update ->
       ?var_update:var_update -> t -> float -> t
update ?size ?mean_update ?var_update t x incorporate x with given size (defaulting to 1) and update rules mean_update, var_update (defaulting to unbiased) into the statistics tracked in t.
val join : ?mean_update:mean_update ->
       ?var_update:var_update -> t -> t -> t
join ?mean_update ?var_update t1 t2 return a Running.t if you had first observed the elements by t1 and then t2 using update rules mean_update and var_update (defaulting to unbiased)
