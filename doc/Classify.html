most_likely probabilities returns the most likely class from the discrete probability distribution.
type ('cls, 'ftr) naive_bayes
A discrete Naive Bayes classifier of class 'cls by observing features 'ftr.
val class_probabilities : ('cls, 'ftr) naive_bayes -> 'cls -> float * float array
class_probabilities bayes class returns the prior and per feature likelihood probabilities learned by bayes for class.
Raises Not_found if bayes never trained on class.
val estimate : ?smoothing:float ->
       ?classes:'cls list ->
       feature_size:int ->
       ('ftr -> int array) ->
       ('cls * 'ftr) list -> ('cls, 'ftr) naive_bayes
estimate smoothing classes feature_size to_feature_array training_data trains a discrete Naive Bayes classifier based on the training_data. to_feature_array maps a feature to an integer array of indices of in the feature_space bounded by [0,feature_size\). Optionally, classes supplies all the classes to learn or they're aggregated from observations in data (this is useful, for classes that may be 'missing' in the data and smoothing is applied.

Additive smoothing can be applied to the final estimates if provided. When estimating a probability distribution by counting observed instances in the feature space we may want to smooth the values, particularly if our training data is sparse.

http://en.wikipedia.org/wiki/Additive_smoothing

val eval : ?bernoulli:bool ->
       ('cls, 'ftr) naive_bayes -> 'ftr -> 'cls probabilities
eval bernoulli classifier feature classifies feature according to classifier. if bernoulli is specified we treat the underlying distribution as Bernoulli (as opposed to Multinomial) and estimate the likelihood with (1-p_i) for features i that are missing from feature.
