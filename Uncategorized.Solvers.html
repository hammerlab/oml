Module Uncategorized.Solvers

module Solvers: sig .. end

Numerically find the roots of passed functions.
val newton_raphson_full : ?init:float ->
       accuracy:float ->
       iterations:int ->
       lower:float ->
       upper:float -> f:(float -> float) -> df:(float -> float) -> float
newton_raphson_full ?init accuracy iterations lower upper f df uses the Newton Raphson method to iteratively solve f.

Starting at mid point of lower and upper, the method uses df to follow the tangent to it's root, that becomes the new point. Iteration continues until f x < accuracy.
Raises Util.IterationFailure if we escape our bounds or take longer than iterations.

val newton : ?init:float -> lower:float -> upper:float -> (float -> float) -> float
newton ?init lower upper f is equivalent to newton_raphson_full ?init ~accuracy:1.0e-10 ~iterations:1000 lower upper f (Estimations.second_order f)
val bisection : epsilon:float -> lower:float -> upper:float -> (float -> float) -> float
bisection epsilon lower upper f iteratively finds the root of f between lower and upper using the bisection method.
Raises Invalid_arg If the function does not take opposite signs at the supplied bounds.
