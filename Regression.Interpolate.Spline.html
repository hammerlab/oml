Module Regression.Interpolate.Spline

module Spline: sig .. end
Cubic splines provide piecewise polynomial to fit the data that is smooth at the fit data points (aka knots). This is achieved by requiring that the polynomials on the two sides of a knot have the same first and second derivative at that point.
type boundary_condition =
| Natural (*
The 2nd derivatives at the end points are 0, y''(x_0) = y''(x_n) = 0 leading to straight lines based off of the cubics fit on the inside.
*)
| Clamped of float * float (*
The 1st derivatives at the end points of the spline are equal to the passed values.
*) 
type t 
val knots : t -> (float * float) array
knots t return the points used to originally fit the spline.
val coefficients : t -> (float * float * float * float) array
coefficients t returns an array of the fit spline coefficients.
val fit : ?bc:boundary_condition ->
       (float * float) array -> t
fit ~bc data
Raises Invalid_argument if data is less than 3 data points.
bc : defaults to Natural
val eval : t -> float -> float
eval spline x evalute spline at x.

Note that if x is outside the original points used to fit the spline, the fit to the closest knot is used.

val eval_arr : t -> float array -> float array
eval_arr t data if data is sorted in increasing order equivalent to Array.map (eval spline) data but a wee bit faster.
